When using this for the first time:

WINDOWS USERS:
1. download Node from nodejs.org and install it.
2. download and install git for Windows from msysgit.github.io. When installing choose the option to run git from the Windows Command Prompt.
2. open up the Command Prompt and install Bower using the node package manager (npm).
	>npm install -g bower
3. from the Command Prompt, again, change directory to the location of micrographIOT.
	>cd path/to/micrographIOT
4. one more time from the Command Prompt...
	>bower install
